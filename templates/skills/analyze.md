# Skill: analyze

## Description
Read-only analysis of existing code, specs, or architecture to surface findings and recommendations.

## Allowed
- Read-only analysis
- Cross-module reasoning
- Cross-project reasoning

## Forbidden
- Code generation
- Code modification
- Refactoring

## Inputs
- AGENTS.md
- specs (optional)
- Source files

## Outputs
- Findings
- Risks
- Observations
- Non-binding recommendations
