# Skill: refactor

## Description
Improve existing code structure without changing behavior, within allowed refactor triggers.

## Allowed
- Refactors that meet Refactor Triggers
- Incremental, scoped improvements

## Forbidden
- Behavior changes
- Architecture changes
- Unrelated modifications

## Inputs
- AGENTS.md
- Specs (optional)
- Source files

## Outputs
- Modified files
- Refactor justification
- Confirmation of unchanged behavior
